新增页面

这里的 “页面” 指代一个已经配置了路由，能够通过 url 直接访问的模块。要新建一个页面通常只需要进行简单的配置。

step 1 新增页面文件夹

在 src/views 下进行页面文件的管理，elite 采用的方法是在 views 目录再新建一层进行页面目录的管理。

例如，在 dashboard 这个布局下的页面全部存储在src/views/dashboard 目录下

在src/views/dashboard 新建 newPage 目录，并且在当前目录新建几个文件：

index.vue

newPage.html

newPage.js

newPage.less

src
assets
components
layout
routes
views
dashboard
newPage
index.vue
newPage.html
newPage.js
newPage.less

然后在index.vue内部建立 vue 文件格式。把外部文件全部引入


<template src="./newPage.html"></template>
<script src="./newPage.js"></script>
<style src="./newPage.less" lang="less" scoped></style>

然后在对应的文件里进行相关的开发即可。

step 2 在路由里加入对应的文件

加入菜单和路由的方式请参照 路由和菜单 中介绍的完成。加好后，访问http://localhost:8080/dashboard/new-page 就可以看到新页面了。
