新增业务组件

对于一些可以被多处引用的功能模块，建议提炼成业务组建统一管理。这些组件一般有以下特征：

  • 只负责一块相对独立、稳定的功能；
  • 没有单独的路由配置；
  • 可能是纯静态的，也有可能包含自己的 data 或 state（vuex），接受其伏组件传递参数控制。

新增组件文件夹

在 src/component 下进行页面文件的管理。

例如，在 dashboard 这个布局下的页面全部存储在src/views/dashboard 目录下

在src/views/dashboard 新建 newComponent 目录，并且在当前目录新建几个文件：

index.vue

newComponent.html

newComponent.js

newComponent.less

src
assets
components
newComponent
index.vue
newComponent.html
newComponent.js
newComponent.less
layout
routes
views

然后在index.vue内部建立 vue 文件格式。把外部文件全部引入


<template src="./newComponent.html"></template>
<script src="./newComponent.js"></script>
<style src="./newComponent.less" lang="less" scoped></style>

然后在对应的文件里进行相关的开发即可。
